# SPLOG - Sistem Pengelolaan Logistik OJK Jawa Timur

Website untuk manajemen inventaris dan pengajuan logistik internal OJK Jawa Timur.

## Fitur Utama

### Mode User
- **Katalog Logistik**: Browse dan cari barang inventaris berdasarkan kategori
- **Keranjang Belanja**: Tambahkan barang ke keranjang dengan kontrol jumlah
- **Riwayat Pengajuan**: Lacak status pengajuan logistik yang telah diajukan

### Mode Admin
- **Dashboard**: Overview statistik dan pending approval requests
- **Manajemen Inventaris**: Update stok barang dan monitor stok kritis
- **Approval System**: Setujui atau tolak pengajuan dari user

## Teknologi yang Digunakan

- React 18
- Tailwind CSS
- Lucide React Icons
- Local Storage (untuk persistensi keranjang)
